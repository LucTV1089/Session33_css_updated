What is CSS ?

Cascading Stype Sheets (CSS) is used to format the layout of a webpage

With CSS, you can control the color, the size of text, the spacing between elements, how elemets are positioned and laid out, what background inmages or background colors are to be used, different displays for differents for different devices and screen sizes, and much more!

Tip: The word cascading means that a style applied elements within the parent. SO, if you set the color of the body text to "blue", all heading, paragraphs, and other text elements within the body will also get the same color(unless you specify something else)
